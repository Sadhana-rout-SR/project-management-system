# TaskFlow - Project Management System

## Overview
TaskFlow is a role-based project management system developed using Flask and MySQL.

## Features
- Admin Dashboard
- Employee Dashboard
- Project Management
- Task Assignment
- Authentication System
- Responsive UI

## Technologies Used
- Python
- Flask
- MySQL
- Bootstrap
- HTML
- CSS

python app.py
